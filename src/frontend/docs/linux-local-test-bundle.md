# Linux Test Bundle Guide

This guide explains how to download, extract, and run the GPIO Control Panel Linux test bundle on any Linux system or Raspberry Pi.

## Overview

The Linux test bundle is a self-contained package that includes:
- Pre-built frontend application
- Compiled backend canister (WASM)
- Launcher script for local deployment
- Raspberry Pi GPIO helper scripts
- Optional systemd service configuration
- Complete documentation
